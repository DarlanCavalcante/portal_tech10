## 🚀 Como Rodar

### Opção 1: Python (Recomendado)

```bash
cd loja_tech_site
python -m http.server 5500
```

### Opção 2: Python3

```bash
cd loja_tech_site
python3 -m http.server 5500
```

### Opção 3: Node.js (serve)

```bash
cd loja_tech_site
npx serve -p 5500
```

### Acessar

**URL:** http://localhost:5500

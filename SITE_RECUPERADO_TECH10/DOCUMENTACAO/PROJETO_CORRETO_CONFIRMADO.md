## 🚀 Como Rodar

### Iniciar Frontend

**Windows (PowerShell):**
```powershell
cd tech10-loja/2710/frontend
python -m http.server 5500
```

**Linux/Mac:**
```bash
cd tech10-loja/2710/frontend
python3 -m http.server 5500
```

**Alternativa (Node.js):**
```bash

```
